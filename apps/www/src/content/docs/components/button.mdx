## Overview

The Button component is designed for maximum visual impact. It utilizes a custom noise background animation and vibrant gradients to create a truly premium feel. It's perfect for primary call-to-actions and hero sections.

## Features

<Callout type="info">
  Key features of the Button component:
  - **Animated Noise**: Built-in noise texture animation for a cinematic texture.
  - **Gradient Themes**: Multiple pre-defined color variants (Default, Emerald, Fire).
  - **Tactile Feedback**: Interactive scaling effects on click/press.
  - **Glassmorphism Influenced**: Leverages subtle shadows and gradients for a glass-like aesthetic.
  - **Highly Responsive**: Seamlessly scales across all device types.
</Callout>

## Variants

### Default
A soft, harmonious mix of pink, blue, and yellow.

### Emerald
A fresh, professional green and cyan theme.

### Fire
A bold, energetic orange, red, and purple theme.
